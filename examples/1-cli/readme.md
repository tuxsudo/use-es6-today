# Babel via CLI


##	Install babel globally

	npm install -g babel


## Transpile Single File

	babel app.js -o dist.js


## Transpile Directory

	babel src -d dist


## Watch and Transpile Changes

	babel src -w -d dist

*	create src/divide.js
*	see what happens



[Reference](https://babeljs.io/docs/usage/cli/)
